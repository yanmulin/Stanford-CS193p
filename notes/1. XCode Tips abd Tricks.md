## Friday Session 1: XCode Tips and Tricks

### Panes: 
1. cmd + 0 
2. cmd + option + 0
3. cmd + <NUM>
4. cmd + option + <NUM>
5. cmd + shift + y

### Assistant Editor: 
1. cmd + option + enter
2. cmd + enter
3. cmd + shift + o
Preference settings: Xcode-Preference-Navigation-Uses Focused Editors
Then, "cmd + shift + o" change the file in the focused Editor

### format the code: 
1. cmd + /
2. cmd + i

### Debug: 
1. po <VAR> (print out object)
2. p <VAR> (print out)
